# 正则表达式的贪婪匹配和最小匹配

## 一、正则表达式的贪婪匹配
re库默认采用贪婪匹配的方式，也就是返回匹配的最长项，如：
```python
>>> import re
>>> i = re.search(r'py.*n','pyanbncndnfngn')
>>> if i:
	i.group(0)
'pyanbncndnfngn'    #默认采用贪婪匹配，返回最长的
```

## 二、正则表达式的最小匹配

只要在的后面增加一个最小匹配操作符，即?问号：

![最小匹配操作符](https://img-blog.csdn.net/20170327211937362)


```python
import re

i = re.search(r'py.*?n','pyanbncndnfngn')
if i:
	print(i.group(0))

pyan  #使用最小匹配操作符?，实现最小匹配
```
