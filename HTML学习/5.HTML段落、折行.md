# HTML段落、折行


## 一、HTML段落：

HTML 可以将文档分割为若干段落。  
段落是通过` <p> `标签定义的  
浏览器会自动地在段落的前后添加空行。(`</p> `是块级元素),格式:  
`<p>这是一个段落</p>`


## 二、折行（换行）：
如果不希望产生一个新段落的情况下进行换行（新行），使用` <br> `标签



## 三、综合应用以上内容：
```html
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <title>我的博客（Yeats.com）</title>
    </head>

    <body>
        <h1>《献钱尚父》</h1> 
        <p>
            贵逼身来不自由，几年勤苦蹈林丘<br><!--在句尾换行-->
            满堂花醉三千客，一剑霜寒十四州<br>
            莱子衣裳宫锦窄，谢公篇咏绮霞羞<br>
            他年名上凌烟阁，岂羡当时万户侯
        </p>
        
    </body>
</html>
```
