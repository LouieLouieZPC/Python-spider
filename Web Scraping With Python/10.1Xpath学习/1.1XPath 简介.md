## 一、简介
XPath 是一门在 XML 文档中查找信息的语言。

XPath 是 XSLT 中的主要元素。

XQuery 和 XPointer 均构建于 XPath 表达式之上

## 二、特点
XPath 使用路径表达式在 XML 文档中进行导航  
XPath 包含一个标准函数库  
XPath 是 XSLT 中的主要元素  
XPath 是一个 W3C 标准
