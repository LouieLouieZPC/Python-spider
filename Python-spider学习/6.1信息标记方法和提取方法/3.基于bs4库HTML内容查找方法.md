# 基于bs4库HTML内容查找方法

以下所有内容基于以下这串代码：
```python
#!/usr/bin/python
# -*- coding: UTF-8 -*-

import requests
from bs4 import BeautifulSoup
url="https://python123.io/ws/demo.html"
r=requests.get(url)
demo=r.text
```
